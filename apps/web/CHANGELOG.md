# Viewer Changelog

## 2026-03-30

- Rebuilt the public station viewer into a desktop-first operational workstation inspired by the NOAA NBM 1D Viewer.
- Added a left control rail for station search, run/member selection, group selection, element browsing, and settings.
- Added stable URL-driven state for chart controls and settings.
- Added click-drag zoom with shared x-axis reset behavior across charts.
- Added ensemble display controls for boxes, whiskers, median, and deterministic overlay.
- Added timezone, theme, alternate color, and font-size controls.
- Added example deep links and expanded documentation for URL parameters.
